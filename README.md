**✅ TodoGo**

A simple and elegant To-Do application built with React and Tailwind CSS. TodoGo helps you stay organized by allowing you to create, edit, and manage your tasks with ease.
<br><br>

**🚀 Features**

✍️ Add, edit, and delete tasks

✅ Mark tasks as completed

🎨 Clean and responsive UI

📦 Data persistence with local storage
<br><br>

**🛠 Tech Stack**

⚛️ React – Frontend framework

🎨 Tailwind CSS – Styling

⚡ Vite – Build tool for fast development
